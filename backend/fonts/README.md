# Fonts for image overlay text (Pillow)

Place DejaVu TTF files here for server-side text rendering on exported images:

- DejaVuSans.ttf
- DejaVuSans-Bold.ttf
- DejaVuSerif.ttf

Download from: https://dejavu-fonts.github.io/

If these files are missing, Pillow falls back to a basic bitmap font (smaller, less polished).
